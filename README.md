# Flowwow Product Images

Репозиторий с изображениями товаров для XML-фида Flowwow.

## Структура

- `images/products/{article}/` - фотографии товаров по артикулам

## Товары

- A06WH - Набор из 6 конфет (японский стиль)
- A07AV - Origami X (лимитированная коллекция, 7 конфет)
- A12WH - Набор из 12 конфет (японский стиль)
- A24WH - Набор из 24 конфет (японский стиль)
